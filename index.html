Política de Privacidad – ClimaSignal

Última actualización: 3 de marzo de 2026

ClimaSignal es una aplicación móvil desarrollada por Green Alert (“nosotros”). Esta política explica qué datos recopilamos, cómo los usamos y qué opciones tienes como usuario.

Si tienes cualquier duda, puedes escribirnos a support@greenalert.app.

1. Responsable del tratamiento

Responsable: Green Alert

Correo de contacto: support@greenalert.app

2. Datos que recopila ClimaSignal

ClimaSignal recopila únicamente los datos necesarios para prestar el servicio de alertas climáticas y mejorar la aplicación.

a) Datos de ubicación

Solo se recogen si concedes el permiso correspondiente al sistema operativo: ubicación aproximada y/o precisa del dispositivo, para mostrar el clima actual y crear y evaluar alertas según tus ubicaciones.

b) Datos de cuenta (cuando creas una cuenta)

  • Correo electrónico.
  • Contraseña (almacenada de forma cifrada).
  • Idioma preferido.
  • En algunos casos, identificadores de acceso con proveedores externos (Google, Facebook) si decides usarlos.

c) Datos de configuración y uso

  • Alertas creadas (tipo de alerta, umbrales, ubicaciones, días de anticipación, horario).
  • Ubicaciones guardadas (nombre y coordenadas).
  • Preferencias de idioma y notificaciones.
  • Información básica de uso de la app de forma agregada (por ejemplo, cuántas alertas tienes activas).

d) Datos del dispositivo

  • Identificador de notificaciones push (token de FCM/Expo).
  • Sistema operativo y versión de la app.

e) Datos de pagos Premium

  • Identificadores de compra proporcionados por Google Play y RevenueCat (por ejemplo, ID de producto, ID de transacción o recibo).
  • No tenemos acceso al número de tu tarjeta ni a datos bancarios completos.

f) Uso sin cuenta (modo anónimo)

Si decides usar ClimaSignal sin crear una cuenta, tus alertas, ubicaciones y preferencias se asocian a un identificador anónimo del dispositivo. Esta información puede sincronizarse opcionalmente con nuestros servidores para poder enviarte notificaciones, pero no se vincula a un correo electrónico u otros datos de identificación directa.

3. Para qué usamos tus datos

Usamos los datos descritos arriba para:

  • Prestar el servicio principal: mostrar el pronóstico del clima, crear y mantener tus alertas, enviar notificaciones cuando se cumplan las condiciones.
  • Notificaciones push: enviar avisos de lluvia, calor u otras condiciones según tus ajustes, y mensajes importantes sobre cambios relevantes en el servicio.
  • Gestión de la versión Premium: verificar y mantener el estado de tu compra Premium (sin anuncios) y evitar fraudes relacionados con compras.
  • Mejorar la app: analizar de forma agregada y anónima siempre que sea posible cómo se utiliza ClimaSignal para mejorar funcionamiento y estabilidad.

No usamos tus datos para elaborar perfiles comerciales avanzados ni para venderlos a terceros.

4. Con quién compartimos tus datos

Compartimos datos solo con proveedores necesarios para el funcionamiento de ClimaSignal:

  • Proveedor de datos meteorológicos: servicios de predicción del tiempo que reciben coordenadas para devolver pronósticos.
  • Notificaciones push (Firebase / Expo / FCM): para enviar notificaciones a tu dispositivo empleando el token de notificaciones.
  • Publicidad (Google AdMob): la versión gratuita muestra anuncios mediante AdMob, que puede recopilar identificadores de dispositivo según sus propias políticas.
  • Pagos in‑app (Google Play y RevenueCat): para gestionar compras de ClimaSignal Premium usando los identificadores de compra necesarios.
  • Infraestructura de backend y base de datos: proveedores de hosting que almacenan de forma segura la API y los datos asociados a tus alertas y cuenta.

No vendemos tus datos personales a terceros.

5. Publicidad y ClimaSignal Premium

La versión gratuita de ClimaSignal muestra anuncios mediante Google AdMob. Puedes optar por la versión ClimaSignal Premium, que elimina los anuncios de la app. Aunque dejes de ver anuncios, seguiremos utilizando algunos datos técnicos para enviar notificaciones, mantener tus alertas y asegurar el correcto funcionamiento del servicio.

6. Base legal y consentimiento

Dependiendo de tu país, la base legal principal para el tratamiento de tus datos puede ser:

  • Ejecución de un contrato: necesitamos tratar ciertos datos (alertas, ubicaciones, cuenta) para poder prestarte el servicio.
  • Consentimiento: obtenemos tu consentimiento expresamente cuando aceptas permisos de ubicación, notificaciones o decides crear una cuenta o realizar una compra Premium.

Puedes revocar algunos consentimientos desde los ajustes de tu dispositivo o contactándonos.

7. Conservación de los datos

  • Conservamos tus datos mientras uses activamente ClimaSignal y mantengas una cuenta con nosotros.
  • Si desinstalas la app y la usas solo en modo anónimo, parte de la información se elimina del dispositivo; los datos asociados a un identificador anónimo se conservarán solo el tiempo necesario para mantener el servicio.
  • Si en el futuro ofrecemos una opción explícita de “Eliminar cuenta”, al usarla borraremos o anonimizaremos los datos personales asociados en un plazo razonable, salvo aquellos que debamos conservar por obligaciones legales.

8. Tus derechos

Dependiendo de tu jurisdicción, puedes tener derechos como acceso, rectificación, supresión, limitación u oposición al tratamiento y portabilidad de tus datos.

Para ejercer estos derechos o plantear cualquier duda, escribe a support@greenalert.app indicando que se trata de una consulta relacionada con ClimaSignal. Es posible que te pidamos información adicional para verificar tu identidad.

9. Seguridad

Tomamos medidas técnicas y organizativas razonables para proteger tus datos, como comunicaciones cifradas mediante HTTPS y acceso restringido a las bases de datos. Ningún sistema es completamente seguro, pero trabajamos para reducir al máximo los riesgos de acceso no autorizado, pérdida o uso indebido de la información.

10. Cambios en esta política

Podemos actualizar esta Política de Privacidad para reflejar cambios en ClimaSignal, en la legislación aplicable o en nuestros procesos internos. Publicaremos la nueva versión en esta misma URL indicando la fecha de “Última actualización”. Si los cambios son significativos, podremos avisarte dentro de la app o por otros medios razonables.

Si tienes cualquier pregunta sobre esta Política de Privacidad o sobre cómo tratamos tus datos en ClimaSignal, por favor contáctanos en support@greenalert.app.
